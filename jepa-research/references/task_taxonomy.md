# JEPA Task Taxonomy (v1)

Use these coarse labels in metadata and answer outputs.

## Modality

- `vision`
- `video`
- `audio`
- `language`
- `multimodal`
- `tabular/time-series`
- `domain-specific`

## Problem Type

- `representation-learning`
- `world-model`
- `planning/control`
- `generation`
- `anomaly-detection`
- `downstream-task-adaptation`

## Evidence Level

- `official` (official paper/repo)
- `reproduction` (community replication)
- `proposal` (new method claim without broad validation)

## Typical Failure Buckets (debug mode)

- `env-dependency`
- `data-contract`
- `shape-mask`
- `optimization-divergence`
- `collapse-degenerate-representation`
- `evaluation-mismatch`
